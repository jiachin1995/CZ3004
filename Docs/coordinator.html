Coordinator¶

class coordinator.Coordinator¶

Coordinator class. Controls the movement of the robot.

fakeRun¶

Defaults to False. Set to True if running in simulation

stepsPerSec¶

Number of steps to take per second, if running in simulation

fakeRunWait()¶

If running in simulation, pause the algorithm for (1/stepsPerSec) seconds.

forward(steps)¶

Moves the robot forward

Parameters

steps – Integer. Number of steps to take.

turnLeft()¶

Turns the robot left

turnRight()¶

Turns the robot right
