Robot¶

class robot.Robot(fakeRun=False, fakeMap=None, stepsPerSec=1, **kwargs)¶

Robot class. Represents the virtual robot.

Orientation refers to where the robot is facing:

  1. Top

  2. Right

  3. Bottom

  4. Left

pos¶

a 15x20 array. Contains None, 0 or 1 as values.

orientation¶

Centre of 3x3 start area. Default = [1,1]

explore¶

if True, update map after every movement

map¶

Map object. Refer to Map.py

sensors¶

Sensors object. Refer to sensors.py

coordinator¶

Coordinator object. Refer to coordinator.py

getBaseLine()¶

baseline refers to the left,middle & right (from the robot’s perspective) tiles immediately in front of robot and the 3x3 space the robot is occupying.

baseline_dict contains the tiles to search. For example, if facing right, search top, middle & bottom tiles

getBaseLineVert()¶

baseline_vert refers to baseline, but vertical. Refer to getBaseLine() above

getTileRange()¶

direction of tiles to search. if facing right, search range of tiles right of robot.

getTileRangeVert()¶

Same as getTileRange() above, but search range of tiles to left of robot
